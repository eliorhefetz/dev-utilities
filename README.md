# dev-utilities

A collection of practical automation, conversion, and developer productivity tools.

## Included tools

- `vcf-to-sql` — convert `.vcf` contact files into SQL `INSERT` statements
- `vcf-to-csv` — convert `.vcf` contact files into CSV rows
- `secret-codec` — encrypt and decrypt text using AES-GCM with a PBKDF2-derived key
- `minify-assets` — minify CSS and JavaScript assets with a single PowerShell script
